Sparrow version 0.2.0

Feb 29, 2016 #sparrow #spark #release

Sparrow version 0.2.0 is now available with updated dependency on Spark 1.6.0.

It’s both available as a Spark package and from the YPG Data Bintray repository.

Release notes

  • Bump Spark version to 1.6.0
  • Test against Scala 2.10.6 on Travis
  • Bump the Macro Paradise plugin to 2.1.0
